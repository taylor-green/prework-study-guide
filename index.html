HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element defines metadata such as the charcater set, description, keywords, author, and viewport.
  • ul is unordered list
  • ol is ordered list
  • li is list items
  • p is paragraph
  • br is a line break
  • The script element embeds client-side scripts in the HTML doc

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Inline CSS is used to style an element directly in the HTML file
  • Internal CSS contains CSS rules for the webpage in the head section of the HTML file.
  • Seperation of concerns is a widely used term that refers to seperating code into different files based on use.
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • API or application programming interface
  • console.log is used to output a message to the web console by adding an ARGUMENT
  • Control flow is the order in which a computer executes code in a script
  • IF statement is a way of introducing decision making into our code
  • array is a single variable that is used to hold a group of data
  • function is a set of instructions that tells the computer how to perform a certain task
